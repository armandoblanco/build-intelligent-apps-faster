Workshop Build Intelligent Apps Faster 

Workshop técnico de 3 horas 

Duración: 3 horas 
Audiencia: Arquitectos, Developers senior, DevOps, líderes técnicos 

Objetivo 

Construir en vivo una aplicación moderna, segura e inteligente, desde el código hasta el despliegue en Azure, usando GitHub, AI y automatización completa. 

 

Estructura del Workshop 

Módulo 1 – Crear una REST API con GitHub Copilot 

Generación asistida de código 

Buenas prácticas de estructura 

Uso efectivo de prompts 

Creación base de la API 

Resultado: API funcional creada en minutos. 

 

Módulo 2 – Agregar inteligencia con AI Foundry 

Integración de capacidades de AI 

Diseño de endpoints inteligentes 

Arquitectura para apps inteligentes 

Resultado: Aplicación tradicional convertida en aplicación con AI integrada. 

 

Módulo 3 – Pruebas y seguridad con GitHub 

Generación de unit tests 

Code scanning 

Secret scanning 

Seguridad integrada en el pipeline 

Resultado: DevSecOps desde el primer commit. 

 

Módulo 4 – Infraestructura como Código con Terraform 

Creación de recursos en Azure 

Azure Container Apps 

Azure Container Registry 

Variables y ambientes 

Resultado: Infraestructura reproducible y gobernada. 

 

Módulo 5 – CI/CD con GitHub Actions 

Pipeline de build y test 

Deploy automatizado a Azure 

Estrategia por ramas 

Protección de environments 

Resultado: Automatización completa del ciclo de vida. 

 

Módulo 6 – Destruir la infraestructura 

Terraform destroy automatizado 

Buenas prácticas de control de costos 

Gestión responsable del entorno 

Resultado: Cultura Cloud madura y controlada. 

 

Resultado final del workshop 

Los asistentes verán en una sola sesión: 

Desarrollo acelerado con AI 

Seguridad integrada 

Infraestructura automatizada 

Despliegue en Azure 

Aplicación inteligente lista para producción
